# prefab.g8
Setup an app + server easily

### Why?

We wanted to make it easy for anyone to set up a new repo for building a phone app.

### Usage

`g8 prefab.g8 {project-name}`

### Add-ons

- Reasonml React Native Client -> `./bin/create_reason_react_native_app.sh`
